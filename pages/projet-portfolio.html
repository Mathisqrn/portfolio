Projet Portfolio Développeur Web

Wireframe / Maquette

Ce portfolio a été conçu dans l’objectif d’illustrer mon travail et de soutenir ma recherche de stage en alternance. Il s’agit d’un projet abouti dans sa première version, mais pensé pour évoluer progressivement en fonction des nouvelles compétences que j’acquerrai au cours de ma formation.

L’intégralité de la maquette a été réalisée sur Figma, incluant la structure, la charte graphique ainsi que les animations prévues pour l’interface. J’ai choisi moi-même le style visuel et les couleurs afin de créer une identité cohérente, moderne et professionnelle. L’objectif était de concevoir une interface sobre mais impactante, orientée recruteur, mettant en valeur mes projets et mes compétences.

Maquette figma de la version téléphone

Développement sur mesure

HTML / CSS

Le site a été entièrement développé en HTML et CSS pur, sans framework, afin de maîtriser pleinement les bases du développement front-end. Une attention particulière a été portée à la structure sémantique du HTML pour garantir un code propre, lisible et optimisé pour le référencement naturel (SEO).

Pour la mise en page, j’ai utilisé Flexbox et CSS Grid selon les besoins : Flexbox pour l’alignement et l’organisation des éléments en ligne, et Grid pour structurer les sections plus complexes comme la présentation des projets. Le CSS est organisé de manière claire et évolutive afin de faciliter les futures améliorations.

JavaScript

Le JavaScript a été utilisé pour enrichir l’expérience utilisateur. J’ai notamment développé une navigation interactive avec animation, ainsi qu’un mode sombre permettant d’améliorer le confort de lecture. Le choix d’utiliser du JavaScript vanilla me permet de comprendre précisément le fonctionnement des interactions et du DOM sans dépendre de bibliothèques externes.

Responsive

Le site a été pensé en desktop-first, puis adapté pour les tablettes et smartphones à l’aide de media queries. L’objectif était de garantir une expérience fluide sur tous les supports, avec une navigation claire et une mise en page réorganisée intelligemment selon la taille de l’écran.

Accessibilité

Une attention particulière a été portée à l’accessibilité : utilisation de balises sémantiques, textes alternatifs pour les images, contrastes de couleurs adaptés et structure logique du contenu. L’objectif est de proposer un site accessible au plus grand nombre tout en respectant les bonnes pratiques du développement web moderne.

Publication GitHub

Le projet est publié sur GitHub, ce qui me permet d’assurer un suivi de version et de faire évoluer le portfolio au fil de mes apprentissages. Cette gestion via Git facilite l’ajout de nouvelles fonctionnalités, l’amélioration du code et la structuration progressive du projet.

Ce portfolio représente aujourd’hui une base solide de mon travail en tant que développeur web front-end. Il reflète à la fois mes compétences actuelles en HTML, CSS et JavaScript, ainsi que ma capacité à concevoir un projet de manière complète : de la maquette à la mise en ligne.
